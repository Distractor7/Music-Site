Process

When it comes to music production. I work with a DAW, which stands for "Digital Audio Workspace". In particular I work with Ableton Live Suite 11. This is a fully integrated studio software that allows you to create complex musical arrangements with both virtual instruments and by running your own hardware through the program. I also typically make use of a midi controller key board and beat pad, which is an external instrument that you can use to create percussion and melodies through. There are many more types of external instruments and midi controllers that can be used in creating music through Ableton Live. I can't wait to get my hands on a few instruments in particular; in time.

When it comes to mixing, I use Serato DJ pro, which is a virtual DJ mixing software usually used in combination with an external DJ controller, CDJ's or another form of music mixing hardware. This software allows you to mix and live edit multiple audio files.

Finally, I also play guitar and drums. These days, I play those instruments for my own enjoyment and havent recorded much with them. But I really woould like to play more and record more instrumentals
